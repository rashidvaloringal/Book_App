# 📚 Islamic Digital Library Pro

PWA-enabled digital library application for reading and listening to Islamic books in Arabic, Malayalam, and English.

## 🚀 Deployment on GitHub Pages

1. Upload all 6 files (`index.html`, `manifest.json`, `sw.js`, `config.js`, `.gitignore`, `README.md`) along with your icons (`icon-192.png`, `icon-512.png`) to your GitHub repository.
2. Go to **Settings** -> **Pages**.
3. Choose **main branch** as the source and click **Save**.
4. Open the generated link in your **Microsoft Edge Browser** on Android and tap **Install App** via the menu button.
